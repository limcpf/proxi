# 무조건 참고할 문서
- `./ARCHITECTURE.md` : 현재 프로젝트의 아키텍처 소개
- `./{각 프로젝트 Repo}/AGENTS.md` : 각 프로젝트의 규칙

# 프로젝트 소개
- 현재 프로젝트 루트는 3개의 프로젝트를 포함한 모노레포이다.
    - 프론트엔드를 맡는 front
    - 백엔드를 맡는 back
    - 그 둘 사이의 공유를 맡는 shared 내부 라이브러리 프로젝트
- 각 프로젝트별 작업시 각 프로젝트의 AGENTS.MD를 참고
- 하나의 PR에 여러 프로젝트 개발이 필요한 경우, Context가 혼동되지 않게 별도의 에이전트로 수행

# 개발 시 규칙
- 항상 Plan을 먼저 세우고 개발에 들어간다.
- 모든 답변과 작업 요약은 한국어로 작성한다.
- 코드 주석은 한국어로 작성한다.
- 사용자가 명시하지 않은 신규 의존성 추가는 피한다.
- 관련 없는 파일은 수정하지 않는다.
- 변경은 가능한 한 최소 범위로 유지한다.
