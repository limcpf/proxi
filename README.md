# proxi

`proxi` 는 `apps/front`, `apps/back`, `apps/shared` 를 분리해 운영하는 pnpm 모노레포다.

## 개발 시작
- 의존성 설치: `corepack pnpm install`
- 개발 환경 구성: [`docs/DEVELOPMENT.md`](./docs/DEVELOPMENT.md)
- 전체 검증: `corepack pnpm run verify`

## 주요 작업 경로
- 백엔드: [`apps/back`](./apps/back)
- 프런트: [`apps/front`](./apps/front)
- 공통 계약: [`apps/shared`](./apps/shared)
- 문서 시스템: [`docs/README.md`](./docs/README.md)
